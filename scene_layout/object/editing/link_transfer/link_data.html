关联数据¶

参考

模式

物体模式

菜单

对象 ‣ 链接/传输数据... 。

将活动对象的场景或数据块之间的对象链接到所有选定对象。在某些情况下（即对象数据，修改器），目标对象必须与活动对象具有相同的类型或能够接收数据。如果目标已经有链接到它们的数据，它将首先被取消链接。

类型

数据块的关联类型。

  • 关联物体数据

  • 关联材质

  • 关联动画数据

  • 关联集合

  • 链接实例集

  • 关联字体到文本

  • 复制修改器

  • 复制蜡笔效果
