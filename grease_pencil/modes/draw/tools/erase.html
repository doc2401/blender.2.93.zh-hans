擦除工具¶

参考

模式

绘制模式

工具

工具栏 ‣ 擦除

此工具擦除已绘制的笔画。

笔刷设置¶

半径

笔刷的半径，单位为像素。

F 允许你通过拖动指针或输入数字然后确认来交互式地改变画笔大小。

使用压感（压感图标）

使用笔尖压力控制效果的强度。

封闭橡皮擦（重叠的方框图标）

仅擦除可见的、不被几何体遮挡的笔画。

默认橡皮擦

Use this brush when enabling the eraser tool with fast switch key (Ctrl).

模式

确定擦除工具的作用方式。

溶解

模拟光栅类的橡皮擦，此类橡皮擦在真正删除点之前会影响笔画的强度和厚度。

强度

控制橡皮擦对描边透明度 （alpha） 的影响程度。

你可以通过在3D视窗中按 Shift-F ，然后移动指针，再按 LMB ，来交互地改变画笔的强度。你也可以用数字来输入尺寸。

使用压感（压感图标）

使用笔尖压力控制效果的强度。

影响笔画强度

擦除时，删除的笔画透明度（alpha）数值。

影响笔画宽度

擦除时，删除的笔画厚度值。

点光

一次性擦除笔画中的点。

笔画

删除整个笔画。

显示游标

Shows the brush shape in the viewport.

用法¶

选择笔刷¶

在 "工具设置" 中，选择要与工具一起使用的画笔。 "擦除" 工具使用 擦除画笔 类型（柔和、尖点和描边）。

消融擦除¶

  • 选择软/硬类型的擦除笔刷。

  • 调整笔刷设置。

  • 点按 LMB 或 Pen 删除视角中的笔画。
