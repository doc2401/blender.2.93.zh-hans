图层列表¶

蜡笔图层面板。¶

蜡笔对象各自具有用于在 列表视图 中分组和排列笔画的2D图层列表。任何笔画只能属于单个2D图层。列表中始终只有一个活动图层（所选图层）。绘制时，新笔触将添加到活动图层。默认情况下，视口中图层的视图顺序是从上到下。

每个图层都对应于摄影表编辑器中的一个通道（在蜡笔模式下）。查看 摄影表 来获得更多信息。

图层也可以与修改器一起使用，仅影响部分绘图。查看 Modifiers 来获得更多信息。

Tip

Sometimes the layers you are not working on can be a distraction. Activate Fade Layers in overlays to control the opacity of the non-active layers. See Overlays for more information.

图层名称旁边有四个图标按钮，用于控制图层的常见属性：

遮罩（遮罩图标）

切换 遮罩 可见性。

洋葱皮（洋葱皮图标）

切换 洋葱皮 时，是否使用该图层。

隐藏 ( 眼睛图标)

在视图和渲染中切换图层的可见性。

锁定（锁上图标）

切换图层不可编辑。

图层专用项

操作项

复制图层

制作所选图层的精确副本，附加数字以区分其名称。

复制空白关键帧

生成选定图层的副本，但关键帧为空。用于轻松预设空关键帧以用于清理或填充过程。

显示全部

打开列表中每个图层的可见性。

隐藏其他

关闭列表中除活动层之外的每个层的可见性。

全部锁定

锁定列表中所有图层的编辑性。

全部解锁

解锁列表中所有图层的编辑性。

淡化非活动几何体

自动锁定列表中除活动图层外的每个图层的版本。这样，您可以避免在其他图层中进行不必要的更改，而无需每次都锁定它们。

禁止锁定材质编辑

避免在图层中编辑锁定的材质。 禁用后，即使它们在材质列表中被锁定，也可以编辑任何材质。

Merge Down

Merge the selected layer with the layer below, the new layer keeps the name of the lower layer.

Copy Layer to Object

Makes a copy of the layer and move it to the selected Grease Pencil object.

可见性（屏幕图标）

切换活动图层是否是唯一可以编辑且可见的图层。

锁定（锁上图标）

切换活动图层是否是唯一可以编辑的图层。

在图层列表下方还有其他常用设置：

混合

要执行混合的操作的图层。见 颜色混合模式 。

不透明度

用于设置图层的不透明度。

使用灯光

启用后，图层会受到光源的影响。

遮罩组¶

在 列表视图 中，受图层遮罩影响的图层。有关更多信息，请参阅 遮罩。

变换¶

Allows per-layers location, rotation and scale transformations.

调整¶

图层调整面板。¶

染色

为图层中使用的任何材质颜色着色的颜色。

系数

控制颜色混合的量。

笔画宽度

覆盖图层中笔画粗细的宽度值。

关系¶

从属/类型

选择父物体，然后键入以操作图层。 该图层将继承父级的变换，这在剪纸动画时非常有用。

索引编号

层编号值可与某些修改器一起使用，以限制仅对某些区域的更改。

更多信息见 修改器。

视图层

定义用于 "蜡笔" 层的 "视图层" 。如果为空，则该图层将包含在所有视图图层中。这对于分离绘图部件 合成 非常有用。

显示¶

自定义通道颜色

设置要在通道区域中使用的颜色 摄影表。

仅在关键帧上显示

仅当视图在实际帧中具有关键帧时，才会使视图在视口中可见。 例如，当您使用 填充 工具进行笔画处理并且只想查看实际帧中的笔画以避免填充不需要的区域时，这会有所帮助。
