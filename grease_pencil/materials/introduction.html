简介¶

材质控制 "蜡笔" 对象的外观。它们定义描边和填充区域的基色和纹理。

列表中始终只有一个活动材质（所选材质）。绘制时，新笔画将使用活动材质。

您可以使用 顶点模式 中的工具或在绘制模式下的绘制和着色工具覆盖基本材质颜色。

材质始终与笔画保持关联，这意味着材质中的任何更改都将更改已绘制笔画的外观。

一些笔画会链接到不同材质。¶

蜡笔着色器¶

蜡笔着色器创建一种材质，该材质可用于 "蜡笔" 对象的描边和/或填充区域。

笔画和填充组件具有自己的分面板，并且可以使用面板顶部的复选框启用它们。

笔画 仅对线起作用， 填充 仅在由封闭线条确定的区域上（通过连接的起点和终点）起作用。

Note

这个着色器目前还不是支持BSDF并能与Blender灯光交互的着色器，只能设置在"材质属性"面板上（它不是着色器节点）。

蜡笔材料可以在 材料属性 中创建，就像Blender中的任何其他材料一样。有关详细信息，请参阅 材质分配 。

3D 视口可以设置为材质预览或渲染着色，以交互方式预览材质在场景中的外观。

蜡笔材质是可以 分配 到一个或多个对象的数据块，并且可以将不同的材质分配给不同的笔画。

在蜡笔中， 画笔 设置以及使用的材料将定义最终笔触的外观和感觉。

列表视图 中的材质槽也有一些额外的控件，有助于在绘制或编辑线条时使用材质。

常规设置¶

蜡笔材料槽面板。¶

在材料名称旁边有三个图标按钮，用于控制材料的常见属性：

锁定（锁上图标）

切换材质是否可编辑。

视图/渲染可见性（眼睛图标）

切换视口和渲染中的材质可见性。

洋葱皮（洋葱皮图标）

切换 洋葱皮 的材质使用。

其他菜单¶

显示全部

打开列表中每个材质的可见性。

隐藏其他

关闭列表中除活动材质外的所有材质的可见性。

全部锁定

锁定列表中所有材料的版本。

全部解锁

解锁列表中所有材料的版本。

未选中项

锁定所选描边中未使用的所有材料。

锁定未使用

锁定列表中所有材料的版本。

删除未使用槽

删除未用到的材质槽

合并相似

组合列表中的类似材质，并将使用合并材质之一的笔触替换为新材质。

转换材质为顶点颜色

Only keeps necessary materials and convert all materials base color to vertex color.

从顶点颜色中提取调色板

Add all used vertex color to a new Color Palette. See 调色板.

锁定和可见性一般控制¶

锁定（锁上图标）

切换当前活动材质是否是唯一可以编辑的材料。

可见性（屏幕图标）

切换当前活动材质是否是唯一可以编辑且可见的材料。

蜡笔着色器¶

蜡笔材质使用特殊的 着色器 来定义描边和填充表面的外观。
