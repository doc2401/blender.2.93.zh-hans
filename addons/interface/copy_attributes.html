复制属性菜单¶

此插件扩展了 复制对象 Ctrl-C 热键，以包括许多额外的复制类型。

激活¶

  • 打开Blender并转到 "首选项" ，然后转到 "附加组件" 选项卡。

  • 单击 界面 ，然后单击 复制属性菜单 前钩上启用脚本。

描述:¶

物体模式¶

选择多个对象，然后按 Ctrl-C 将属性从活动对象复制到选定对象，您将看到以下菜单：

菜单上的每个项目都将从 活动 （最后选定对象） 复制到所有其他 选定 项的某些属性：

复制位置

复制在全局坐标中物体所处的位置。

复制旋转

复制在全局坐标中物体所处的旋转。

复制缩放

将对象缩放复制到世界坐标中。

复制绘制（视图显示）选项

复制显示选项：例如线框、空显示形状、空显示大小等。

Copy Time Offset

Copies the animation stack time offset.

复制实例化

复制对象的实例化 组/顶点/面 设置。

(Self-explanatory)

复制保护的变换

复制对象约束

将删除所选对象上的所有约束，并将其替换为活动对象上的约束。

网格编辑模式¶

选择网格物体后，进入 "编辑模式" ，然后按 Ctrl-C 键复制纹理面属性。 出现以下菜单：

Some of the items may not appear, depending on the number of UV texture or vertex color layers on the mesh.

The first group of options copies attributes from the active face to all other selected faces in the same UV texture or vertex color layer.

复制材质

将材质索引复制到选定的面。

复制图片

复制图像分配到选定的面。

复制UV坐标

将UV坐标复制到选定的面。

复制顶点色

Copy Vertex Colors to selected faces.

The second group of options copies attributes to selected faces in the active UV texture or vertex color layer from their corresponding faces in a different UV texture or vertex color layer.

Copy Image from Layer

Copy image assignment from another layer for selected faces.

从图层复制UV坐标

复制UV从另一层为选定面坐标。

Copy Vertex Colors from Layer

Copy Vertex Colors from another layer for selected faces.

姿态模式¶

选择多个骨骼，按 Ctrl-C 键将属性从 active*（最后选择的骨骼）复制到所有其他 *选中 骨骼。出现以下菜单：

复制本地位置

将位置坐标（如在变换面板中看到的）复制到选定的骨骼。

复制本地旋转

将旋转坐标（如在变换面板中所示）复制到选定的骨骼。

复制本地缩放

将缩放坐标（如在变换面板中看到的）复制到选定的骨骼。

复制视觉位置

将骨骼的实际位置（如屏幕所看到的）复制到选定的骨骼。

复制视觉旋转

将骨骼的实际旋转（如屏幕所看到的）复制到选定的骨骼。

复制视觉缩放

将骨骼的实际缩放（如屏幕所看到的）复制到选定的骨骼。

复制骨骼形状

(Self-explanatory)

复制保护的变换

(Self-explanatory)

复制姿态约束

(Self-explanatory)

复制IK限制

复制姿态

在安装插件之前最初映射到 Ctrl-C 。

本地与视觉之间的区别¶

骨骼的局部变形是相对于每个骨骼自己的静止位置而言的。 如果它们在 "编辑" 模式下不匹配，则即使在骨骼的 "变换" 面板中数字看起来相同，它们也不会在 "姿势" 模式下匹配。 他们也没有考虑约束或亲子关系。

骨骼的可视化变换将活动骨骼的视觉变换（在屏幕上看到的）复制到选定的骨骼，而不管亲子关系或约束条件如何。即使骨骼的数量不匹配，结果也应该看起来完全相同。选定骨骼的约束可能会阻止此它起作用（或与此相关的驱动）：
