蜡笔工具¶

一套用于绘制蜡笔的工具。

激活¶

  • 打开Blender并转到 "首选项" ，然后转到 "附加组件" 选项卡。

  • 单击 "对象" ，然后单击 "蜡笔工具" 以启用脚本。

工具¶

盒子变形¶

围绕蜡笔笔画创建变形框。按 Ctrl-T 启动变形模式。

操作范围取决于模式：

  • 对象模式：整个蜡笔对象变形。

  • 编辑模式：变形所选点。

  • 绘制模式：仅变形最后一次笔画。

变形过程中提供的快捷方式：

  • Spacebar / Return 以确认。

  • Delete/ Backspace/ Ctrl-T/ Tab (两次)来退出。

  • M 在线性和样条线模式之间进行切换。

  • 1-9 来设置盒子的细分级别。

  • Ctrl-Left/ Ctrl-Right 在X轴上逐步细分方框，在Y轴上 Ctrl-Up/ Ctrl-Down 。

旋转画布¶

在自由导航或相机视图中的活动相机中执行视图的旋转。

  • 维护 Ctrl-Alt-MMB 以旋转视图（可在加载项首选项中自定义）。

  • Click and release immediately to reset view (free navigation only).

拉直¶

拉直第一点与最后一点之间的笔画。可以在重做面板中调整影响。

示波器

  • 在蜡笔绘制模式下的最后一次笔画。

  • 在蜡笔编辑模式下选择笔画。

Tip

直线影响重置

影响百分比存储用于下一次使用。使用按钮上的 Shift-LMB 将影响重置为完全。

刷包安装程序¶

安装包括蜡笔 纹理刷包 （由Daniel Martinez Lara制作）。这在 画笔面板 的绘制模式下可用。

Note

This feature will be removed once the Blender official asset manager is released.
