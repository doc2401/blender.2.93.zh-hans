编辑网格工具¶

网格编辑工具为Blender增加了几个内置工具中没有的工具，或者为类似任务提供不同的方法。下列每个菜单：顶点、边、面和实用工具是在 侧边栏 ‣ Edit 标签 的一个子面板，默认是关闭的。关闭的面板标题栏中的图标包含了一些新的/不同的选择工具和编辑模式顶点/边/面的选择模式，用于在不打开子面板时快速访问一些工具。

在UI中，每个类别中都有各种工具，大多数工具都有一个工具提示，提供一些使用信息，帮助用户了解每个工具的基本要求。'最重要的热键是在编辑模式下双击 RMB 点击后会根据顶点/边/面的选择模式调出顶点/边/面的菜单。菜单集成是将主要工具集成到编辑模式的上下文菜单中，选择工具集成到编辑模式的选择菜单中。

  • Face Inset Fillet：完全基于zmj100的插件

  • Vertex Align: 作者 zmj100

  • Edge Fillet Plus: 作者 Gert De Roost -- -- 原创作者：zmj100

  • Split Solidify: 作者 zmj100, 由zeffii更新为BMesh

  • Pen Tool: 作者 zmj100

  • Mesh Cut Faces：作者 Stanislav Blinov (Stan Pancakes)

  • Vertex/Edge/Face Context Menu：作者 Stanislav Blinov (Stan Pancakes)

  • Edge Roundifier: 作者 Piotr Komisarczyk (Komi3D), PKHG

  • PKHG Face Extrude: 由PKHG提供，基于 Geodesic Domes 插件的 面功能

  • Set Edges Length：作者 Giuseppe De Marco [BlenderLab] 灵感来自NirenYang

  • Edge Tools: 作者 Paul Marshall (brikbot)

  • Extrude and Reshape：作者 Germano Cavalcante (mano-wii)

激活¶

  • 打开Blender并转到 "首选项" ，然后转到 "附加组件" 选项卡。

  • 选择 Mesh 然后选择 Edit Mesh Tools 以启用插件。

挤出和重塑¶

您可以选择一个面，然后将其向内或向外挤压，创建新的面以适应挤压。注：也称为 推/拉面 。
