VR Scene Inspection¶

VR 场景检测插件 在用户界面中发布并扩展了Blender的本地虚拟现实功能。该功能集仅限于场景检测使用案例。通过Blender内部的进一步开发，可以实现更高级的使用案例。

Blender中的VR支持基于OpenXR规范，需要一些设置步骤。这些在 头戴式显示器 (HMD) 部分有解释。

激活¶

  • 打开Blender并转到 "首选项" ，然后转到 "附加组件" 选项卡。

  • 点击3D视图，然后点击VR场景检查来启用脚本。

界面¶

位于 3D 视图‣ 侧栏 ‣ VR 标签 。

VR 会话¶

开始VR会话

尝试建立与OpenXR平台的连接，与一个 HMD（头戴式显示器） 共享视口。

Positional Tracking

只跟踪头部的旋转变化，不允许HMD影响观看者在虚拟空间的位置。

视图¶

基面

在VR视图中设置地平面的可见性。

标注

在VR视图中设置注释笔画的可见性。

裁剪起点/结束点

Clipping values of the VR view, as in the 3D Viewport.

地标¶

Landmarks are used to store reusable base poses (position and rotation) for the viewer in the virtual space.

地标

列表视图 。

已选中地标

定义哪个地标的设置会显示在列表下面。改变选定的地标对VR视图没有影响。

Activate (star icon)

激活一个地标，使其改变VR视图的基本姿态。

添加 +

创建一个地标

移除 -

Deletes the selected landmark.

类型
场景相机

Follow the scene's active camera to define the base pose of the viewer.

自定义

Set an arbitrary camera to define the base pose of the viewer.

视口反馈¶

显示VR相机

在当前 3D 视口中加入一个指示图标，展示当前 VR 查看器的姿势（在虚拟空间中的位置和旋转）。

镜像VR会话

使当前的3D视口跟随VR视图的视角。
