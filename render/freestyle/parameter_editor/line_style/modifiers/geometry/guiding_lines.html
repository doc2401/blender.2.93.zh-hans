引导线¶

引导线 修改器将描边替换为连接描边两端的直线。

偏移量

沿原始描边偏移起点和终点，然后生成新的直线笔触。

当描边足够短时，此修改器将产生合理的结果，因为较短的描边更有可能被直线很好地近似。因此，建议将此修改器与 Strokes 面板中的拆分选项之一（按 2D 角度或按 2D 长度）一起使用。
