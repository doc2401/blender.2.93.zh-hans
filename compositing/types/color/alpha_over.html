Alpha上叠节点¶

Alpha上叠节点。¶

Alpha Over 节点用于将图像彼此叠加。如果前景图像像素的字母大于0 ，则它将覆盖在背景图像上。

输入¶

系数

控制前景图像的透明程度。小于 1 的参数将使前景图像变得更加透明。

图像

背景 图像输入端。

图像

前景 图像的输入。

属性¶

转换预乘

将前景图像转换为 预乘Alpha 格式。

Alpha Over* 节点被设计为与预乘的alpha颜色格式一起工作。当你知道你的图像有 直通型Alpha 颜色值时，使用这个复选框来执行正确的over操作。结果仍然是预乘的阿尔法。

预乘

使用 预乘Alpha 或 直通型Alpha 之间进行混合。

当设置为1时，前景色值将乘以alpha，即预乘;这相当于启用 转换预乘法 。设置为 0 时，颜色值不会更改。

如果 预乘 不为零，则 转换预乘 将被忽略。

Note

这是一个旧版选项。

输出¶

图像

标准化图像输出。
