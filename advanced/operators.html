启用 Developer Extras

在文本编辑器中创建一个文本文件，提供Python语法中所有运算符及其默认值的列表以及生成的文档。这是获取所有Blender操作员概览的好方法。

重载脚本¶

参考

模式

所有模式

菜单

Topbar ‣ Blender‣系统‣重新加载脚本

快捷键

Ctrl-Alt-T

重新加载脚本数据文件夹中的所有脚本；文本编辑器中的任何更改都将丢失！

内存统计¶

参考

模式

--debug-memory

菜单

Topbar ‣ Blender‣系统内‣存统计信息

可以通过使用 运算符搜索 搜索 "来搜索" 内存统计信息 "来找到该运算符，该运算符将打印有关内存对象，其大小和用户计数的有用信息。

Important

要使用该操作的完整功能，请从控制台使用 --debug-memory 启动Blender。

调试菜单¶

参考

模式

所有模式

菜单

Topbar ‣ Blender‣系统‣调试菜单

快捷键

Ctrl-Alt-D

该操作将弹出一个菜单，用于设置Blender为某个调试模式。

See the source code for a description of what each value does.

Tip

开发人员可以在代码中搜索 G.debug_value，查找该操作的其他用法。

Note

可以在调试模式下启动Blender或设置 bpy.app.debug = True 来访问其他调试选项。

重绘计时器¶

参考

模式

所有模式

菜单

Topbar ‣ Blender ‣ System ‣ Redraw Timer

快捷键

Ctrl-Alt-T

该操作将显示一个菜单，包含一组对撤销/重做功能的基准UI渲染时间的测试。
