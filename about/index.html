贡献文档¶

Blender手册是一个社区驱动，人人皆可参与的项目。无论你想要修复一个小小的拼写错误，或重写整个章节，都非常欢迎你来帮助Blender手册建设！

If you find an error in the documentation, please report the problem.

通过项目的 联系方式 参与讨论。
