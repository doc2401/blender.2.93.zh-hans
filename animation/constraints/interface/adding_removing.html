添加/移除约束¶

关于本页所描述的物体约束使用方法也可以用在骨骼约束中。

选项卡¶

参考

模式

物体模式

菜单

属性编辑器（Properties) ‣ 物体选项卡(Object tab) ‣ 集合(Collections）

若要添加约束在Constraints（约束）选项卡点击 Add Object Constraint（添加对象约束） 菜单。

移除约束 点击在 标题栏 右边的``X`` 按钮。

菜单¶

添加约束（带目标 ）¶

参考

模式

物体模式和姿态模式

菜单

对象 ‣ 约束 ‣ 添加约束（带目标）

向活动对象添加约束。约束类型必须从弹出菜单中选择，但稍后可以从 添加约束（带目标） 调整上一步操作 面板进行更改。如果除了活动对象之外还选择了其他对象，则该对象将成为约束目标（如果所选约束接受目标）。

姿势模式下，当使用另一个骨架的骨骼作为约束的目标时，该工具将查看非活动骨架内部并使用其活动骨骼。

将约束复制到当前所选¶

参考

模式

物体模式和姿态模式

菜单

物体 ‣ 约束 ‣ 将约束复制到当前所选

将激活物体的约束复制给非激活的选择物体

清除物体约束¶

参考

模式

物体模式和姿态模式

面板

物体 ‣ 约束 ‣ 清除物体约束

Removes all Constraints of the selected object(s).

追踪¶

参考

模式

物体模式

面板

物体 ‣ 跟踪

为非激活的选择物体添加约束，目标对象是激活选择物体。而激活选择物体不会被添加约束。

  • 阻尼跟踪约束(Damped Track Constraint)

  • 标准跟踪约束(Track To Constraint)

  • 锁定跟踪约束(Lock Track Constraint)

Clear Track 清除跟踪

从选定的对象中删除所有阻尼跟踪，标准跟踪和锁定跟踪约束。

清除跟踪并保持当前变换（取消跟随）

移除选定对象的所有跟踪约束，并保持由跟踪引起的最终变换。
