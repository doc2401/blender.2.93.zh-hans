驱动器编辑器¶

驱动编辑器允许用户使用另一个属性驱动一个属性。见 驱动器 与 函数曲线.

驱动器编辑器。¶

主要区域¶

The main view allows you to view and edit Driver F-curves. An F-curve has several key parts:

轴向

曲线定义了两个属性之间的关系：当前（驱动）属性（Y轴）和驱动（X轴）。

参见 函数曲线。

控制柄

驱动曲线上的每个点都有一个控制柄，帮助确定两个值之间的关系。它们可以被选择和修改来改变曲线的形状。

See 控制柄类型（Handle Type） for more information.

一个简单的驱动。¶

See also

请参阅 函数曲线 了解更多信息。

导航¶

与大多数编辑器一样，您可以：

平移 MMB

点击并拖动 MMB，垂直（值）或水平（时间）平移视图。

缩放 Wheel

用鼠标滚轮放大和缩小.

缩放视图 Ctrl-MMB

缩放垂直或水平视图。

查看所有 Home

在可用的空间内适应曲线。

此外，您还可以使用滚动条来平移和缩放视图。

标题栏¶

视图控制¶

视图控制。¶

规格化

规格化曲线，使最大和最小点分别等于1.0和-1.0。

自动

自动重新计算每条曲线编辑的曲线标准化。这对于防止曲线在调整后跳跃很有用。

仅显示所选（鼠标光标图标）

只包含与所选物体和数据相关的曲线。

显示隐藏（虚线物体图标）

包含来自不可见的物体/骨骼的曲线。

仅显示错误（警告三角图标）

仅包括已禁用或有错误的曲线和驱动。 对调试很有用。

创建幽灵曲线（带有曲线图标的正方形）

在编辑器的背景中显示所选曲线的当前状态快照。这对于有一个比较的基础，在此基础上进行编辑很有用。

过滤器（漏斗图标）
类型过滤

按属性类型过滤曲线。

数据块排序（az图标）

物体数据块按字母顺序显示，以便更容易找到它们发生的位置（以及帮助将相关对象的动画在一起）。

此选项可能会影响重场景的回放速度。
