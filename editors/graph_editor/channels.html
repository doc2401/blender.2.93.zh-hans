通道¶

通道区¶

通道区。¶

通道区域用于选择和管理曲线编辑器的曲线。该部分将物体及其动画数据层次结构以标题形式显示。每个级别可以通过标题左侧的小箭头展开/折叠。

  • 场景，物体（深蓝色）

  • 动作， 形态键，等（浅蓝色）

  • 群组（绿色）

  • 通道（灰色）

Name Filter

仅显示与搜索文字匹配的通道。按下反转按钮将显示除与搜索文本匹配的通道以外的所有通道。

装配器¶

在标题栏上，有控制通道的设置的切换控件：

固定（图钉图标）

Make the channel always visible regardless of the current selection (Graph editor only).

隐藏 ( 眼睛图标)

Hides the channel(s)/curve (Graph editor only).

修改器（扳手图标）

Deactivates the F-curve modifiers of the selected curve or all curves in the channel.

Mute (speaker icon)

禁用通道/曲线。

锁定(挂锁图标) Tab

切换可编辑状态下的通道/曲线。所选通道可以通过按下 Tab 来锁定。

Note

In the Dope Sheet this is also working inside the NLA, but that it does not prevent edition of the underlying F-curve.

选择¶

  • 选择通道（文本以白色/黑色）: LMB

  • 多重选择/取消选择: Shift-LMB

  • 全选 A

  • 取消全选 Alt-A

  • 框选：（LMB 拖动）或 B (LMB 拖动)

  • 取消框选：（Ctrl-LMB 拖动）或 B (Shift-LMB 拖动)

  • 选择通道中的所有关键帧：双击 LMB 通道标题栏。

编辑¶

  • 重命名: Ctrl-LMB

  • 删除选择: X 或 Delete

  • 锁定所选: Tab

  • 启用通道设置: Shift-Ctrl-W

  • 禁用通道设置: Alt-W

  • 切换通道设置: Shift-W

滑块¶

显示滑块的动作编辑器。¶

在通道标题栏上，你可以使用另一个包含数字按钮或滑块的列，从而允许你更改当前关键帧的值或添加新的关键帧。有关如何显示这些滑块，请参阅 视图菜单 。

编辑¶

删除通道¶

参考

菜单

删除通道

快捷键

X

Deletes the whole channel from the current action (i.e. unlink the underlying F-curve data-block from this action data-block).

Warning

X 快捷方式是与区域相关的： 如果你在左列表部分使用它，它将删除选定的通道，而如果你在主区域中使用它，则会删除选定的关键帧。

控制添加通道到的群组¶

参考

菜单

频道 ‣ 下/组频道

快捷键

Ctrl-Alt-G, Ctrl-G

取消/组合所选通道到可以通过双击组名以重命名的集合中。 例如，这有助于组合与骨架的一部分关联的通道，以使编辑器更有条理。

参考

菜单

快捷键

显示/隐藏¶

隐藏所选曲线 H

隐藏所选曲线。

隐藏隐藏未选项 Shift-H

仅显示所选曲线（并隐藏其他所有内容）。

显示曲线 Alt-H

显示所有先前隐藏的曲线。
