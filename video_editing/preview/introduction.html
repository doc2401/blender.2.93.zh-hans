简介¶

序列编辑器预览用于显示序列编辑器时间轴的渲染结果。这可进一步配置为来自确定的通道，叠加或图像分析器（范围）的输出显示。可以 NumpadPlus 放大视图， NumpadMinus 缩小视图。

标题栏¶

序列编辑器标题栏。¶

视图菜单¶

侧栏 N

显示或隐藏 侧栏 。

工具栏 T

显示或隐藏 工具栏 。

变换过程中预览

在转换剪辑的起始/结束句柄时显示起始/结束框架的预览。

适应窗口预览 Home

调整预览的大小，使其适合该区域。

缩放 Shift-B

点击并拖动绘制一个矩形，然后会缩放到这个矩形大小。

分（倍）数缩放

从1:8到8:1逐步调整预览大小。

序列渲染图像

Render the an image at the current frame.

序列渲染动画

从时间轴的起始帧到结束帧，将预览渲染为一个视频文件或者图像序列。

导出字幕

Exports Text strips, which can act as subtitles, to a SubRip file (.srt). The exported file contains all Text strips in the video sequence.

切换序列编辑器/预览 Ctrl-Tab

在序列编辑器和预览之间切换编辑器显示类型。

显示模式¶

Mode to show different aspects of the composite result, for the current frame:

图像预览

Render image preview.

亮度波形

Brightness/contrast analyzer.

色度矢量示波图

Color hue and saturation analyzer.

直方图

RGB distribution histogram.

显示通道¶

Color and Alpha

通过棋盘模式使用透明度来显示预览图像。

颜色

忽视预览图像的透明度（所有透明区域将不显示）。

覆盖切换¶

Overlays are information that is displayed on top of the preview region. There is a switch to turn off/on all overlays for the preview region.

预览叠加层

帧叠加

显示 帧叠加，用于将当前帧与参考帧进行比较。

安全区域

在预览上显示叠加层，标记标题安全区域的位置。

元数据

Display Image metadata in the preview area.

标注

在预览区域中显示 标注 。

小部件¶

在序列编辑器预览区域使用装配器来移动和缩放图像。
