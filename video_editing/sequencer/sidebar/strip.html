片段¶

标题栏¶

类型

图标表示片段类型。

名称

可以在此处命名或重命名片段。

屏蔽

If checked the strip will not produce any output.

合成¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

混合

片段与下方通道的混合模式。

不透明度

设置片段的不透明度（alpha）。当动画化并且 视图 ‣ 显示函数曲线 选项启用时，这个值会在片段上绘制一个跟随动画曲线的暗部。

变换¶

参考

面板

侧栏（Sidebar) ‣ 变换(Transform）

起始位置 X/Y

用于沿 X 轴和 Y 轴移动帧。

X缩放，Y缩放

在X轴和Y轴上缩放图像。

旋转

沿Z轴二维旋转输入。

镜像

沿着X轴（从左到右）或Y轴（从上到下）镜像图像。

裁剪¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

用于裁剪源图像。使用 顶部 、 左 、 底部 和 向右 来控制裁剪的像素数。

视频¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

闪烁

每隔n帧显示。例如，如果你设置此项为10，则片段只会显示源视频的1，11，21，31，41帧。 闪烁 是一个浮点值 -- 使用这种方法，你可得到一个与节拍同步的闪烁效果，例如，使用非整数值。

反转帧序

从序列的最后一帧倒序播放片段。

颜色¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

饱和度

增加或减少图像的饱和度。

正片叠底（相乘）

使用此数值乘以颜色。这会增加亮度。

转换为浮点数

转换输入为浮点数据。

声音¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

体积光渲染

声音的音量。当动画并激活: 查看 ‣ 显示F-曲线 选项时，此值作为跟随动画曲线的暗色部分绘制在条上。该值也反映在波形中。

音调

播放速度的系数。该值会影响片段的长度，但不会时间轴里显示。

平移

Used to pan the audio from left and right channels. Only works for mono sources. Values can be between -2 and 2, where 0 means front/center, -1 means to the left and 1 to the right. In case of multichannel audio (rear speakers) you can pan to those with the higher values: -2, 2 is back. So this value basically represents the angle at which it's played.

显示波形图

Display an approximate waveform of the sound file inside of the sound strip. The waveform reflects strip volume and its animation using keyframes.

单声道

将所有音频通道混合为一个。

时间¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

时间面板用于控制片段源及其时间轴位置。

锁定（锁上图标）

防止片段被移动（可在面板标题栏中找到）。

通道

改变片段的通道编号。

起始

改变片段的起始帧编号，这与选择再移动片段的效果一样。

持续时间

改变片段的长度。这效果也可通过改变结束帧来实现，与选择再移动片段的右控制柄的效果一样。

结束

指定片段的结束时间和结束帧号。

片段偏移量起始/结束

用于通过重复最后一帧将片段延伸以至超出结束帧。或者它可用于缩短片段，就如裁剪结束帧。这与调整片段控制柄的效果一样。

硬切偏移量起始/结束

未剪切片段内容的偏移量。

当前帧

Position of the Playhead relative to the start of the active strip.

源¶

参考

面板

侧栏（Sidebar) ‣ 片段(Strip) ‣ 活动片段(Active Strip）

源面板用于控制片段源，例如文件名称、文件路径和解释这些文件的各种方法。

路径

包含源文件的目录。当文件被移动时，可更新这里而不用重新创建片段了。

文档

源文件的文件名称。对于图像序列中的图像片段，每一帧的名称都不相同。

改变数据/文件

与 路径 和 文件 项相同，但这会同时打开文件浏览器，以便找到搜索的文件。与 片段 ‣ 输入 ‣ 改变路径/文件 相同。

MPEG预搜索

仅适用于视频 -- 使用预搜索项以告知Blender基于指定的之前的帧数（例如MPEG-2 DVD的15帧）来回看并合成图像。

色彩空间

指定源文件的色彩空间。

Alpha模式

如果源文件有一个Alpha（透明）通道，你可选择：

直通型Alpha 或 预乘Alpha

流编号

仅适用于影片 -- 对于具有多个影片流的文件，使用具有给定目录的流。

反交错

删除视频文件中的字段。 例如，如果它是模拟视频并且它具有偶数或奇数隔行扫描字段。

分辨率

活动片段图像输出的分辨率。

声音片段的选项¶

声音

数据块菜单 选择声音。

路径

使用 数据块 菜单的声音文件的路径。

打包（Pack）

将声音打包进blend文件。

缓存

解码声音文件并加载进内存。
