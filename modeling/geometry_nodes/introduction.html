简介¶

Nodes can be used to change an object's geometry in a more complex way than regular modifiers. To use them add a Geometry Nodes Modifier.

修改器堆栈中几何节点修改器的属性。¶

Node Categories¶

属性

用于处理每个对象元素存储的数据的节点，例如，顶点组。

颜色

用于修改颜色数据的节点通过颜色套接字。

几何数据（Geometry）

可以在不同几何类型（体积、网格）上运行的节点。

输入

输入节点为其它节点提供输入数据。

网格

仅操作活动的骨骼

Mesh Primitives

创建原始网格的节点，例如一个立方体。

点光

Nodes that modify the object elements, e.g. vertices.

实用工具

Nodes with general purpose for modifying data.

矢量

矢量节点

体积光渲染

用于创建或使用卷的节点。

Node Data Previews¶

The data values for Attributes are shown in the Spreadsheet editor. You can also preview the attributes up to a certain node by toggling the node preview with the button to the right of the node label.
