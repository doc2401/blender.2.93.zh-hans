Custom Data¶

Sculpt Mask¶

清除雕刻遮罩数据

Removes the sculpting mask data layer. See 清除雕刻遮罩数据 for more information.

Skin Data¶

添加/清除蒙皮数据

Used to manage the skin data layer which is used by the 蒙皮修改器. This operator can be needed in case a Skin modifier is created but no skin data exist.

自定义拆边法向¶

Add Custom Split Normals Data

Adds a custom split normals data layer, if none exists yet.
