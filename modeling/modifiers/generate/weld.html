焊接修改器¶

焊接 修改器查找阈值内的顶点组并合并它们，从而折叠周围的几何体。

选项¶

焊接修改器。¶

模式

用于选择要合并的顶点的方法。

全部

合并包括所有几何形状，包括松散的部件。

Connected 相连项

合并仅包括附加的几何体，即修改器不会将松散的部分合并在一起。

距离

顶点必须相互合并的最大距离。

Duplicate Limit

For a better performance, limits the number of elements found per vertex. 0 makes it infinite.

顶点组

选择 顶点组 选项时，只有权重高于零的顶点才会受到修改器的影响。

反转 <->

反转所选顶点组的影响，这意味着该组现在表示不会被修改器改变形状的顶点。

反转顶点组的权重值。
